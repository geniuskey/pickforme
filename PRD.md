# PickForMe - Product Requirements Document

## 1. 개요

### 1.1 제품명
**PickForMe** (픽포미)

### 1.2 URL
`pickforme.kr` (또는 `pickforme.vercel.app` MVP 단계)

### 1.3 한 줄 설명
"나에게 맞는 제품 찾기" - 심리테스트 형식의 질문에 답하면 개인화된 상품 추천 TOP 10을 제공하는 쇼핑 큐레이션 서비스

### 1.4 핵심 가치
- **재미**: 심리테스트처럼 즐기는 쇼핑 경험
- **개인화**: 같은 카테고리라도 사용자마다 다른 결과
- **신뢰**: 니즈 기반 추천으로 높은 구매 전환율

---

## 2. 타겟 사용자

### 2.1 Primary
- 제품 선택에 어려움을 느끼는 쇼핑 초보자
- 심리테스트/MBTI 등 자기 분석 컨텐츠를 좋아하는 20-40대

### 2.2 Secondary
- 선물 고르기가 어려운 사람
- 제품 비교에 시간 쓰기 싫은 실용주의자

---

## 3. 핵심 기능

### 3.1 MVP (Phase 1)

#### 3.1.1 메인 페이지
- 카테고리 목록 (카드 형태)
- "나에게 맞는 _____ 찾기" 형식
- 인기 테스트 순위

#### 3.1.2 테스트 페이지
- 4~8개의 Yes/No 또는 2지선다 질문
- 진행률 표시 (프로그레스 바)
- 뒤로가기 가능
- 애니메이션 트랜지션

#### 3.1.3 결과 페이지
- 사용자 타입 (예: "귀차니즘 집순이")
- 타입 설명 (2-3문장)
- 맞춤 추천 TOP 10 제품 리스트
  - 제품 이미지
  - 제품명
  - 가격
  - 추천 이유 (1문장)
  - 쿠팡 파트너스 링크
- SNS 공유 버튼 (카카오톡, 트위터, 링크 복사)

#### 3.1.4 자동 컨텐츠 생성
- Gemini 3.0 Flash API로 신규 카테고리 자동 생성
  - 질문 세트 생성
  - 제품 태깅
  - 결과 타입 & 설명 생성
- 하루 1-2개 신규 카테고리 자동 게시

### 3.2 Phase 2 (추후)
- 회원가입/로그인
- 결과 히스토리 저장
- 결과 이미지 자동 생성 (SNS 공유용)
- Google AdSense 연동

### 3.3 Phase 3 (추후)
- 리뷰 분석 기반 추천
- 가격 변동 알림
- 네이버/11번가 등 멀티 플랫폼

## 5. 성공 지표 (KPI)

### MVP 단계
- DAU: 100+
- 테스트 완료율: 70%+
- 쿠팡 링크 클릭률: 30%+
- 평균 체류시간: 3분+

### 성장 단계
- 월 방문자: 10,000+
- 쿠팡 파트너스 월 수익: 10만원+
- SNS 공유 수: 일 50건+
